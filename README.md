# Customer Retention and Sales Analysis using Cohort Analysis on Online Retail Data

## Project Overview
This project analyzes over 541,000 online retail transactions to identify sales trends, customer behavior, and retention patterns.

## Objectives
- Clean and preprocess retail transaction data
- Perform exploratory data analysis
- Analyze customer purchasing behavior
- Conduct cohort analysis
- Generate business insights

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
- GitHub
